Checkout Redesign

Dive into the entire process involved in creating this checkout fully responsive and compliant with WCAG standards, ensuring accessibility across devices.

Project Overview

This project aimed to optimize the checkout experience for mobile users. Key changes included making the layout responsive, using a 100vw container, eliminating pixel-based dimensions, and creating a scalable, accessible design that adjusts fluidly to different screen sizes as well as integrating Pay Pal Pay in 4 and Apple Pay

Research & Problem Identification

BrowserStack testing and user feedback revealed issues with hard-coded pixel dimensions, a lack of scalability, and usability challenges on small screens. Addressing these issues required a complete overhaul of the layout to support flexible and responsive design principles.
